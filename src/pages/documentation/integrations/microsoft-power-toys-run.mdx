## Microsoft PowerToys run

As described in [their website](https://learn.microsoft.com/en-us/windows/powertoys/run), **Microsoft PowerToys run** is a quick launcher for Windows power users.

You can add this [third party plugin](https://github.com/inukiwi/PowerToysRun-Shlink) from [inukiwi](https://github.com/inukiwi) to generate short urls using your own Shlink instance.

![Microsoft PowerToys run Shlink plugin](/images/integrations/microsoft-power-toys-run.png)

You can find installation and usage instructions in their [README](https://github.com/inukiwi/PowerToysRun-Shlink?tab=readme-ov-file#powertoys-run-shlink-url-shortener).
